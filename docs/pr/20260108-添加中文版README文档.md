# 添加中文版 README 文档

> 创建 README.zh-CN.md，让中文用户能够更好地理解和使用 Pi Agent。

## 背景与目的 (Why)

Pi Agent 是一个企业级 AI Agent 系统，虽然已有英文版 README，但考虑到：
1. 项目主要用户群体可能包含中文用户
2. SYSTEM.md 中明确说明"用户输出用中文"
3. 提供双语文档可以提升项目的可访问性

本次变更的目的是创建中文版 README，保持与英文版相同的内容结构和质量，让中文用户能够无障碍地理解项目的设计哲学、架构原则和使用方法。

## 变更内容概述 (What)

### 主要变更点：

1. **新增 README.zh-CN.md 文件**
   - 完整翻译英文版 README.md 的所有章节
   - 保持章节结构完全一致
   - 技术术语保留英文（SSOT、Token、Mermaid 等）
   - 代码示例保持不变

2. **翻译策略**
   - 保持专业性和术语一致性
   - 所有链接路径保持不变
   - Mermaid 图表语法保持不变
   - 中文字符密度更高，文件更简洁

   - 风险：文档链接路径可能错误
   - 缓解：保持与英文版相同的路径
