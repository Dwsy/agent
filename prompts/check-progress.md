---
description: 查询项目完成情况 - 帮助用户回顾任务进度和实现状态
---

使用 subagent 工具检查以下内容的进度：$@

1. 首先搜索关于 "$@" 的相关上下文：
   - 在 docs/issues/ 目录中搜索匹配的 issue
   - 在 docs/pr/ 目录中搜索匹配的 PR
   - 使用代码搜索（fd/rg）查找相关的代码实现
   - 检查与 "$@" 相关的 TODO/FIXME 注释

2. 然后分析发现并提供完整的状态报告，包括：
   - 任务是否已完成、进行中或未开始
   - 相关的 Issue/PR 文档及其状态
   - 代码实现位置和完成度
   - 任何待处理工作或阻塞项
   - 如果未完成，下一步行动

3. 输出格式：
   ```
   任务: $@
   状态: [已完成 | 进行中 | 未开始]
   
   相关文档:
   - [Issue/PR 名称] - [状态]
   
   实现情况:
   - [文件路径和位置]
   
   备注:
   - [关键发现、阻塞项或待处理事项]
   
   下一步:
   - [如果未完成，列出行动项]
   ```

作为单个 agent 任务执行，包含完整的上下文收集和分析。
