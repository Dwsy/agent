# 游戏列表

本目录包含6个经典终端游戏和1个实用工具，每个游戏都支持暂停/恢复、状态保存等功能。

## 🎮 可用游戏

### 1. Snake（贪吃蛇）
**命令**: `/game:snake`

**玩法**:
- 使用方向键或 WASD 控制蛇移动
- 吃到食物得分，蛇身变长
- 防止撞墙或撞到自己

**控制**:
- ↑↓←→ 或 WASD: 移动
- P: 暂停（可恢复）
- ESC: 保存并退出
- Q: 退出不保存
- R: 重新开始（游戏结束时）

**特性**:
- 高分记录
- 状态保存和恢复
- 平滑的移动体验

## 🛠️ 实用工具

### Model Test（模型测试）
**命令**: `/game:modeltest`

**功能**:
- 同时测试多个大语言模型的响应速度和质量
- 实时显示测试进度和结果
- 智能处理长文本，防止超出终端边界

**控制**:
- ↑↓ 或 J/K: 导航结果列表
- Enter 或 Space: 查看详细信息
- P: 暂停/恢复测试
- ESC: 保存并退出
- Q: 退出不保存
- R: 重新开始（测试完成后）

**特性**:
- 并发测试（最多5个同时）
- 实时进度条
- 状态图标显示（✅成功 ❌失败 ⏱️超时 ⏳运行中 ⏸️等待）
- 智能文本省略（长模型名、长输出）
- 详细视图查看完整输出
- 测试结果排序和筛选
- 状态保存和恢复
- 响应式布局适应不同终端宽度

**测试模型**:
- modelscope/Qwen3-VL-235B-A22B-Instruct
- nvidia/minimax-m2.1
- nvidia/qwen3-coder-480b-a35b-instruct
- nvidia/z-ai/glm4.7
- proxypal/deepseek-r1
- proxypal/glm-4.7
- proxypal/kimi-k2-thinking
- proxypal/minimax-m2.1
- proxypal/qwen3-coder-plus
- proxypal/qwen3-max
- x-aio/GLM-4.7
- x-aio/MiniMax-M2.1
- x-aio/XAIO-G-3-Flash-Preview
- xiaomimimo/mimo-v2-flash

**技术细节**:
- 并发限制: 5个模型同时测试
- 超时设置: 30秒
- 测试问题: "用一句话介绍你自己，不超过50字。"
- 最小终端宽度: 80字符

**详细文档**: [Parallel Test README](../parallel-test-tui/README.md)

## 🎯 通用特性

所有游戏都支持以下功能：

### 暂停/恢复
- 按 `P` 键暂停游戏
- 暂停后按任意键恢复
- 游戏画面保持不变

### 状态保存
- 按 `ESC` 键保存当前状态并退出
- 下次启动时自动恢复
- 高分记录持久化

### 退出选项
- 按 `ESC` 保存并退出
- 按 `Q` 退出不保存（清除状态）

### 重新开始
- 游戏结束时按 `R` 重新开始
- 保留高分记录
