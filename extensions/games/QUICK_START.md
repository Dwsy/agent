# 🎮 终端游戏框架 - 快速开始指南

## 🚀 快速开始

### 启动游戏

```bash
# 贪吃蛇
/game:snake

# 俄罗斯方块
/game:tetris

# 2048数字游戏
/game:2048

# 扫雷
/game:minesweeper

# 打砖块
/game:breakout

# 乒乓球
/game:pong
```

---

## 🎯 通用控制

所有游戏都支持以下控制：

| 按键 | 功能 |
|------|------|
| P | 暂停游戏 |
| 任意键 | 恢复游戏（暂停时） |
| ESC | 保存并退出 |
| Q | 退出不保存 |
| R | 重新开始（游戏结束时） |

---

## 🎮 游戏特定控制

### Snake（贪吃蛇）
- ↑↓←→ 或 WASD: 移动

### Tetris（俄罗斯方块）
- ←↓→: 移动
- ↑: 旋转
- Space: 快速下落

### 2048（数字游戏）
- ↑↓←→ 或 WASD: 移动方块

### Minesweeper（扫雷）
- 点击: 揭示方块

### Breakout（打砖块）
- ←→ 或 AD: 移动挡板

### Pong（乒乓球）
- ↑↓ 或 WS: 移动挡板

## 💡 提示

1. **暂停游戏**: 按 `P` 键随时暂停，不会丢失进度
2. **保存退出**: 按 `ESC` 保存当前状态，下次自动恢复
3. **高分记录**: 所有游戏都会保存你的最高分
4. **重新开始**: 游戏结束后按 `R` 快速开始新游戏
